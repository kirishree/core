What is new in Sensei 0.7.0-rc2
  • Google applications browsed via Chrome are now being identified (QUIC over UDP protocol support).
  • Data retirement option introduced. With this option you can define how long to keep your reports (days)
  • Mobile devices support
  • Reports enhancement: Drill-down in Security reports is now available
  • Reports enhancement: Daily executive reports. Selected reports delivered via a daily e-mail.
  • You can easily add block/allow rules within Sensei's Session Explorer based on Application and Application Category.
  • User's Manual in English.
  • Changelog between updates can be viewed
  • Fixed Rebellion Theme compatibility issues.
  • Sensei speaks your language now, we added i18n support to match your OPNsense UI language. English and German are there for now, more coming soon.
  • 350+ new applications identified.
  • First Premium Subscription features are introduced
  • License can be activated from the UI now
  • More performance & stability improvements.
