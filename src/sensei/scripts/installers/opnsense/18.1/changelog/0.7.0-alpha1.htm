What is new in Sensei 0.7.0-release
  • Data retirement option introduced. With this option you can define how long to keep your IPDRs.
  • Changelog between updates can be viewed by user.
  • Drill-down in alert reports is now available.
  • User's Manual in English.
  • You can easily add block/allow rules within Sensei's Session Explorer based on Application and Application Category.
  • Reports enhancement: Daily executive reports. Selected reports delivered via a daily e-mail.
  • 350+ new applications identified.
  • More performance & stability improvements.
